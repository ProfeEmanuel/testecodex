# Organizador de Rotina

Uma página simples para organizar tarefas da semana em formato de tabela:

- **Linhas:** dias da semana (segunda a domingo)
- **Colunas:** horários

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Clique em cada célula e escreva a tarefa desejada.
3. A rotina é salva automaticamente no navegador (localStorage).
